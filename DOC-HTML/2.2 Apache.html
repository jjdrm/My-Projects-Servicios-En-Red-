Manual sencer Apache2 (versió httpd): MANUAL

Rule no. 1: don’t panic! The list before you might seem intimidating at the moment, but that is simply because you are not yet familiar with Apache.

INSTAL·LACIÓ

Ho feim amb l’instucció # sudo apt-get install apache2 abans ens convé actualitzar el repositori usant l’instrucció
# sudo apt-get update

A la versió Ubuntu on posa httpd ha de posar apache2.

EXERCICIS

En quina carpeta es troben els logs?

/var/log/apache

Quins tipus de fitxer log trobam dins la carpeta anterior?

El access.log, el error.log i other_vhosts_acces.log

Volem saber les vegades que s’ha accedit des d'una IP del vostre ordinador d’aula. Quina instrucció usarem?

Anar a la carpeta del server.pid i quin PID té el servidor. Posa les instruccions.

Es PID és 884.

Quines instruccions s’usen per posar el servidor en marxa, per aturar, per saber l'estat i per reiniciar?

  • Marxa: vagrant up
  • Aturar: vagrant stop/halt
  • Estat: vagrant status

Fer una còpia de l’index.html del servidor, posali nom index-copia.html

Edita un document index.html i posa el teu nom.
